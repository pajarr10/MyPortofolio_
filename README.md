## TECH STACK

### Frontend

```
  HTML5          ->  Struktur halaman utama dan bio-link
  CSS3 Vanilla   ->  Custom Properties, Flexbox, Grid, animasi
  JavaScript     ->  ES6+, Vanilla — modal, scroll, counter, parallax
  Tailwind v4    ->  Khusus halaman Bio Link via CDN browser
  Google Fonts   ->  Space Grotesk, Poppins, Space Mono, JetBrains Mono
  SVG Sprite     ->  Ikon inline SVG, zero dependensi eksternal
```

### Backend / Analytics

```
  Node.js >=18   ->  Runtime backend analytics
  Express.js     ->  REST API: tracking pengunjung & admin dashboard
  Upstash Redis  ->  Penyimpanan data analytics (cloud)
  JSON Storage   ->  Fallback lokal jika Redis tidak tersedia
```

### DevOps

```
  Vercel         ->  Hosting: static file + serverless function
  vercel.json    ->  Routing: assets statis + API endpoint
```

## INSTALASI LOKAL

### Prasyarat

```
  Browser modern    -> Chrome, Firefox, Edge, Safari
  Node.js >= 18     -> Hanya jika mau jalankan analytics backend
```

### 1. Clone Repository

```bash
git clone https://github.com/pajarr10/MyPortofolio_.git
cd MyPortofolio_
```

### 2. Buka Frontend (Tidak perlu build)

```bash
# Opsi A — Python
python3 -m http.server 3000

# Opsi B — Node.js npx
npx serve .

# Opsi C — VS Code
# Klik kanan index.html -> Open with Live Server
```

Buka `http://localhost:3000` di browser.

### 3. Jalankan Analytics Backend (Opsional)

```bash
cd analytics
npm install
node server.js
```

Endpoint yang tersedia:

```
  POST  /api/visit   ->  Kirim data kunjungan
  GET   /api/stats   ->  Ambil statistik pengunjung
  GET   /admin.html  ->  Dashboard admin
```

### 4. Environment Redis (Opsional)

Buat `.env` di dalam folder `analytics/`:

```env
UPSTASH_REDIS_REST_URL=https://your-url.upstash.io
UPSTASH_REDIS_REST_TOKEN=your-token-here
```

Tanpa ini, data otomatis disimpan ke `analytics/data/visit.json`.
